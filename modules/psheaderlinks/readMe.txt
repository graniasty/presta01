#########################  Configurable Header Links - Prestashop MODULE  #########################

= Installation =

1. Copy the content of the downloaded zip file in the modules folder, or upload the entire zip from your back office
2. open the Front Office Features Tab and install "Configurable Header Links"
3. For an optimal display, go to modules -> positions, look for the 'Top of Pages' Hook, and bring 'Configurable header links' at the top (see included screenshot "position")
4. Add links from the module's configuration page
